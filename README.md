# CLI Sokoban

This is a command-line Sokoban puzzle game written in F# for the CS20200 term project.

## Project Overview

The player pushes boxes onto goal tiles in a grid-based puzzle map.

## Planned Features

- ASCII-based board rendering
- Player movement using W, A, S, D
- Box-pushing logic
- Stage clear condition
- Move counter
- Restart option
- Undo function if time allows
